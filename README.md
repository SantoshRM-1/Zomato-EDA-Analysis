# Multi Dataset EDA Analysis

## Overview

This repository contains Exploratory Data Analysis (EDA) performed on multiple real-world datasets. The objective of this project is to understand data patterns, perform data cleaning, apply feature engineering techniques, and extract meaningful insights.

---

## Datasets Covered

### Zomato Dataset

* Analysis of restaurant data
* Exploration of cuisines, ratings, and locations
* Identification of popular restaurant trends

### Flight Price Dataset

* Analysis of flight ticket pricing
* Feature engineering on date, time, and duration
* Study of the impact of stops, airline, and journey timing on price

### Black Friday Dataset

* Analysis of customer purchase behavior
* Exploration of demographic factors such as age, gender, and occupation
* Identification of high-value customer segments

---

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## Key Steps Performed

* Data Cleaning (handling missing values and formatting)
* Feature Engineering (date extraction, time processing, duration conversion)
* Encoding (Label Encoding and One-Hot Encoding)
* Data Visualization
* Insight Extraction

---

## Key Insights

* Flight prices increase with number of stops and duration

* Certain airlines consistently have higher ticket prices

* Majority of flights fall within specific time ranges

* Zomato data highlights dominant cuisines and popular locations

* Ratings distribution shows most restaurants fall within a mid-range

* Black Friday analysis indicates the 26–35 age group has the highest spending

* Male customers tend to have higher purchase amounts

* Product categories significantly influence purchase behavior

## How to Run

1. Clone the repository:

```
git clone https://github.com/SantoshRM-1/multi-dataset-eda-analysis.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:

```
jupyter notebook
```

4. Open and run the notebooks inside their respective folders
